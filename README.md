# sub09
課題09・画像認識じゃんけんアプリ
アプリ上で手の画像を認識し、コンピュータ側が絶対に勝つじゃんけんアプリです。
自分の手の画像（グー・チョキ・パーをそれぞれ5枚ずつ）をscikit-learnで機械学習させ、モデルを作成しました。モデルを用いて、撮影した手がグー・チョキ・パーのどの形に当てはまるかを予測します。コンピュータ側は、予測を基に勝つための手の形を選びます。そして、手のイラストを画面上に表示します。
イラスト素材：いらすとや
